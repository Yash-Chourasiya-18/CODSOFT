# 🔢 Calculator App

This is a simple calculator application built using **HTML, CSS, and JavaScript** as part of the **CodSoft Internship Task 3**.

## 📌 Features

- Basic Arithmetic Operations: Addition, Subtraction, Multiplication, Division
- Responsive UI
- Clear Button to reset inputs
- Keyboard input support (optional, if added)

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
